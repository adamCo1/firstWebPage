Difference between web1.0 , web2.0 and web3.0 :

Web 1.0 Web 2.0 Web 3.0
Web 1.0 described the "readable" phrase . in this version , the interaction between the user and the web site was very limited , as to only enabling the user to recieve information , without the abillity to "give" information , such as posts, reviews,comments, or anything really. Web 2.0 describes the "writeable" phrase. In this version , the interaction between users and the web was expanded , so the user can now also "give" information , not like in Web 1.0 . It encourages information sharing and user's participation . Web 3.0 is the "excecuteable" phrase . In this version , the dynamic applications were introduced to the web , which enables interactive services such as HTML5 tutorials , and other many useful applications which makes the web even more interaction encouraging than ever before.
